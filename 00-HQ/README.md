# ZEN0SEC — Cybersecurity HQ

Master index and control surface for all ZEN0SEC knowledge, protocols, playbooks, and lab operations.

## Directory Structure

| Directory | Purpose |
|---|---|
| `00-HQ/` | Master index, ethics charter, operator profile, mission statement |
| `01-methodology/` | NIST SP 800-115, PTES, OWASP WSTG alignment docs and testing protocols |
| `02-lab-design/` | Lab network architecture, OS setup, VM topology, isolation specs |

| `05-linux/` | Linux pentest, privilege escalation, living-off-the-land |
| `06-web-and-api/` | Web attacks, SQLi, IDOR, JWT, WAF bypass, Burp Suite, OWASP WSTG |
| `07-wireless-and-ssh/` | Wi-Fi hacking, SSH access and pentesting |
| `08-hardware-and-memory/` | Rowhammer research: GPU, TRR, Blacksmith, survey, solving |
| `09-soc-and-ir/` | SOC/IR playbook, detection engineering, purple team |
| `10-certifications-and-training/` | OSCP cheat sheet, pentest guides, bug bounty, beginner guides |
| `11-tools-and-references/` | LoLBAS, curl, Docker pentest, DDoS, cheat sheets, web references |
| `12-dev-and-scripting/` | Bash reference, ReactJS cheat sheet, scripting resources |
| `PLAYBOOK/` | Living operational playbook — all protocols, ROE, engagement docs |
