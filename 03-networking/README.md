## Key Tools

- `nmap`, `masscan` — host/port discovery
- `wireshark`, `tshark` — packet capture and analysis
- `netdiscover` — ARP-based host discovery
- `tcpdump` — CLI packet capture
- `hping3` — custom packet crafting

## Lab Scenarios

- Map lab segments with nmap, document all live hosts and services
- Capture AD authentication traffic with Wireshark on Segment B
- Identify service banners and match to CVEs
- Practice reading PCAP files and identifying attack patterns
