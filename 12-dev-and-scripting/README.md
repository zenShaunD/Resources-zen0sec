## Scripting Languages for Security

| Language | Best Use Cases | Priority |
|---|---|---|
| **Python** | Automation, exploit dev, scripting, parsers, APIs | 🔴 Primary |
| **Bash** | System automation, one-liners, tool wrappers | 🔴 Primary |
| **PowerShell** | Windows automation, AD scripting, post-exploitation | 🟡 Secondary |
| **Ruby** | Metasploit module development | 🟡 Secondary |
| **Go** | Fast CLI tools, network tools (e.g., gobuster, nuclei) | 🟢 Learn as needed |
| **Rust** | High-performance low-level tools | 🟢 Learn as needed |
